推荐接入方式

OpenWebUI、Claude Desktop 转发器、OpenCode 等客户端优先填写 OpenAI Gateway Base URL，模型名用 local-current。需要给局域网设备使用时，先进入对应“服务后台”打开局域网提供服务，再创建独立客户端 API Key。

文档与排错
